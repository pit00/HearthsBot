# HearthsBot
Source code of Hearthstone Telegram Bot write in Node.js

## Description
Fetch information from Hearthstone game (minions, spells, ...)

## Link
For more information and how to use, access:
[@HearthsBot](http://telegram.me/HearthsBot)
